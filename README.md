This project predicts the risk of disease recurrence using healthcare data. The objective is to build a predictive analytics pipeline that supports early intervention and better decision-making.

- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

## Workflow

1. Data loading and cleaning
2. Exploratory data analysis
3. Feature engineering
4. Train-test split
5. Model training
6. Model evaluation

## Results

- Accuracy: 97.37%
- Strong classification performance
- Low prediction error
